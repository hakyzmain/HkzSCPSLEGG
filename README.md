# HkzSCPSLEGG

Egg для Pterodactyl: **SCP: Secret Laboratory** + **EXILED**.

**Author:** hakyzscpsl@gmail.com

## Установка

1. Admin → Nests → **Import Egg**
2. Загрузите `HkzSCPSLEGG.json`
3. Создайте сервер → **Reinstall**

## Переменные

| Переменная | Значение по умолчанию |
|------------|------------------------|
| Установить EXILED | `true` |
| Версия EXILED | `false` (последний stable) |
| Бета-ветка Steam | `none` |

## Файлы

- `HkzSCPSLEGG.json` — egg для импорта в панель
- `install.sh` — скрипт установки
